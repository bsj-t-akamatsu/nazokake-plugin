# なぞかけ収集

ネット上からうまいなぞかけを探して、コレクションに追加します。

## 手順

1. **検索**: $ARGUMENTS が指定されていればそのテーマで、なければ幅広くなぞかけを検索する。
2. **選別**: 掛かり方がきれいで意外性のあるものを選ぶ。既に `${CLAUDE_SKILL_DIR}/../nazokake/examples.md` にあるものは除外する。
3. **追加**: examples.md の適切なカテゴリに追加する。カテゴリがなければ新設する。
4. **報告**: 追加したなぞかけの一覧をユーザーに見せる。

## 選別基準

- AとBが遠い関係で意外性がある
- 同音異義語の掛かり方が自然
- 短くてキレがある
- 下品・差別的でない

## 出典

追加時は出典URLをコメントとして残す。
